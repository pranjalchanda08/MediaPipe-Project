# OpenCV Hand Tracking
A project using OpenCV and Mediapipe library to track the hand landmarks.

## System Requirements
* Python 3.2 - 3.7
* No GPU requirements

## Installation
```sh
pip3 install -r requirements.txt
```
## Run
```sh
python3 main.py
```
